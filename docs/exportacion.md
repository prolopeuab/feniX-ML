# 3. Exportación

En este paso generas el XML-TEI final a partir de los archivos ya cargados y revisados.

## Exportar XML-TEI

Para exportar el XML:

1. (Opcional) Define ruta y nombre en `Archivo`.
2. Pulsa `Generar archivo XML-TEI`.

## Nombre y ruta de salida

- Si defines ruta manual, la app guarda ahí el archivo `.xml`.
- Si no defines ruta, la app genera el nombre automáticamente a partir del título y guarda en el directorio de ejecución.

## Antes de exportar

> - `Prólogo y comedia` cargado.
> - Validación ejecutada y revisada.
> - `Vista previa (XML)` o `Vista previa (HTML)` revisada.
> - `Tabla de metadatos` cargada si necesitas un `teiHeader` completo.
